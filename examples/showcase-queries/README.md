# Job & Chain Queries Showcase

Demonstrates read-only query methods for inspecting job chains and jobs.

## Scenarios

1. **Single Lookups**: Get a job chain or job by ID with type narrowing
2. **Paginated Lists**: Filter and paginate job chains and jobs
3. **Chain Jobs**: List jobs within a specific chain ordered by chain index
4. **Blocker Queries**: Inspect blocker relationships from both directions

## Running

```bash
pnpm install
pnpm start
```
